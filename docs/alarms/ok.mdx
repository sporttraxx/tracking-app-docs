# OK

The green OK button allows you to send an OK alarm to race control.

## Sending an OK

1. Press and hold the green OK button in the right side of the [main page](../main_page).

   <div style={{ textAlign: "center", padding: 30 }}>
     <img src={ok_button} width={100} />
   </div>

2. A countdown will be displayed, keep holding the button.

<ScreenshotOSTabs page="alarms" img="timer-ok" width={300} />

3. You can either specify a reason for the OK alarm or tap outside and go back to the main interface.

<ScreenshotOSTabs page="alarms" img="ok-choices" width={300} />
