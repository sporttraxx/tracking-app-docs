# RED FLAG

Il RED FLAG è l’unico allare trasmesso dalla direzione gara al
concorrente.

<ScreenshotOSTabs page="alarms" img="redflag" width={300} />

## Ricevere un RED FLAG

Quando si riceve un allarme RED FLAG, un pop-up rosso viene
mostrato nella parte superiore dello schermo. Premi il tasto
_Acknowledge_ per confermarne la ricezione alla direzione gara.
