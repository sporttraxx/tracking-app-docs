# OK

Il pulsante verde OK permette di trasmettere alla direzione gara un
messaggio OK.

## Trasmettere un OK

1. Premi e tieni premuto il pulsante verde OK nella parte destra della [pagina principale](../main_page).

   <div style={{ textAlign: "center", padding: 30 }}>
     <img src={ok_button} width={100} />
   </div>

2. Un countdown viene mostrato, tieni premuto il pulsante.

<ScreenshotOSTabs page="alarms" img="timer-ok" width={300} />

3. Ora puoi specificare uno stato oppure uscire e ritornare alla
   sezione precedente.

<ScreenshotOSTabs page="alarms" img="ok-choices" width={300} />
