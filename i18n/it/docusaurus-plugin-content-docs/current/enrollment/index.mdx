# Registrazione

Al fine di registrarti all’evento ti serve una chiave alfanumerica che ti
viene fornita dall’organizzatore. Questo codice è unico e serve ad
associare il tuo smartphone al tuo ruolo nell’evento.

## Primo utilizzo/ non ancora registrato

1. Hai due diverse opzioni: registrazione con scansione di un QR code o input manuale di una chiave alfanumerica

<ScreenshotOSTabs page="enrollment" img="start" width={300} />

2. Inserisci la tua chiave o scansiona il QR code. Premi _Enroll_.

<ScreenshotOSTabs page="enrollment" img="key" width={300} />

3. Ti verrà chiesto di confermare i tuoi dati. Verificali attentamente e, se
   corretti, premi _Confirm_. Altrimenti premi _Cancel_ per ritornare alla schermata precedente.

<ScreenshotOSTabs page="enrollment" img="confirm" width={300} />

4. La procedura di registrazione è ora terminata e il sistema ha
   registarto il tuo smartphone come un tracker. Verrai indirizzato alla
   [pagina principale](../main_page).

## Già registrato

Se il tuo Smartphone è già registrato come tracker in un evento, verrai
indirizzato direttamente alla [pagina principale](../main_page).
